## Cómo agregar un nuevo boletín

1. Crear una carpeta nueva: `boletines/especial-02/`
2. Colocar el archivo HTML del boletín como `index.html` dentro de esa carpeta
3. Actualizar `index.html` de la raíz para agregar el nuevo boletín al archivo
4. Subir los archivos al repositorio
